# Contributing

DBM Design System is a solo-maintained project. The repository is public and the published packages are MIT licensed, so you're welcome to use, fork, and modify the code under the terms of that license.

However, **this repository does not accept external contributions.** Pull requests, however well-intentioned, will not be reviewed or merged. There is no contributor workflow, no CLA, no review process to document — this is intentional, not an oversight.

If you find a bug or have a feature request, feel free to open an issue for visibility, but please don't expect a response or a fix on any particular timeline.

If you'd like to build on this project, forking under the MIT license is the right path — not a pull request here.
